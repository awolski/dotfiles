* Install on another system

  #+begin_src shell
  echo 'alias config="/usr/bin/git --git-dir=$HOME/.cfg/ --work-tree=$HOME"'
  echo ".cfg" >> .gitignore
  git clone --bare https://www.github.com/username/repo.git $HOME/.cfg
  config checkout
  config config --local status.showUntrackedFiles no
  #+end_src
